# What This Bot Does Not Build

This lane does not build a general Polymarket bot. It does not chase news, infer probabilities for arbitrary markets, or optimize for event-sniping. It does not trade sports, politics, geopolitics, placeholder outcomes, or augmented neg-risk placeholders.

It does not share runtime, inventory, state, or secrets with `crypto-bot` or `stocks-bot`. It does not bypass geoblock constraints, and it does not promote itself to live trading because a backtest looked pretty.

If a market requires narrative interpretation over operationally clear rules, V1 does not trade it.
